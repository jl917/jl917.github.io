# Embedding

机器学习中Embedding是什么


### 검색

- TF-IDF(Term Frequency-Inverse Document Frequency) 단어 빈도와 역 문서 빈도를 곱하여 특정 문서에서 단어의 중요도를 나타내는 통계적 측정 방법입니다.
- BM25(Best Matching 25) 검색어와 문서 간의 관련성을 계산해 가장 적합한 문서를 찾아주는 검색 랭킹 알고리즘이다.
